# 天気予報時系列バーチャート作成ツール

夜勤明けの引継ぎ資料向けに、都道府県・地域別の時系列バーチャートをブラウザで作成するためのツールです。

## できること

- 都道府県ごとに表示パターンを切替
- 地域ごとに要素行を追加
- セルの値入力
- セル背景色のドラッグ一括変更
- 範囲指定による囲み枠作成
- セル右上への注意報マーク表示
- ブラウザ保存
- バーチャート部分のPNG出力

## ファイル構成

- `index.html`
- `style.css`
- `app.js`
- `README.md`

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作る
2. 上記4ファイルをアップロードする
3. GitHubの `Settings` → `Pages` を開く
4. `Deploy from a branch` を選ぶ
5. Branchを `main` / `/root` に設定して保存する
6. 数分後に公開URLが発行される

## 使い方

### 値入力
- 編集モードを「値入力」にする
- セルをクリックして値を入力する

### セル色変更
- 編集モードを「セル色変更」にする
- 色を選ぶ
- セル上をドラッグして範囲指定する

### 囲み枠作成
- 編集モードを「囲み枠作成」にする
- 枠色と必要ならラベルを設定する
- セル上をドラッグして範囲指定する

### 注意報マーク
- 編集モードを「注意報マーク」にする
- `発` / `解` / `発解` を選ぶ
- セル上をドラッグして範囲指定する

### 保存
- 保存名を入力して保存ボタンを押す
- 保存データはブラウザの `localStorage` に保存される

### PNG出力
- PNGファイル名を入力する
- PNG保存ボタンを押す
